# Foobar

Foobar is a Python library for dealing with word pluralization.

- [DEMO on Heroku](https://afternoon-woodland-62064.herokuapp.com/)

### Install

```sh
$ make install
```

### Run

```sh
$ make start
# open http://localhost:5000
```
